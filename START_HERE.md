## 🚀 Deploy in 1 Click

### Option 1: Vercel (Easiest)
Click "Publish" button in v0 interface

### Option 2: Docker
```bash
docker build -t retailanalytics .
docker run -p 3000:3000 retailanalytics
```

### Option 3: Node.js Server
```bash
pnpm build
pnpm start
```

See [DEPLOYMENT.md](./DEPLOYMENT.md) for full options.
